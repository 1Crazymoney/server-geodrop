# GeoDrop API

A REST API server for the GeoDrop.cash game. Forked from this [koa-api-boilerplate](https://github.com/christroutner/koa-api-boilerplate)

## Install
```
git clone https://github.com/GeoDrop-cash/server-geodrop
cd server-geodrop
npm install
```

## Usage
```
npm run docs
npm start
```
